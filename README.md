# Easen Post — Instagram自動投稿アプリ

Easenネイルサロン向けのInstagram・X自動投稿PWAアプリです。

## 機能
- 写真の一括アップロード・自動スケジュール生成
- AIによるキャプション・ハッシュタグ自動生成
- Sacramento フォントによるロゴ透かし（ゴールド→ピンクグラデーション）
- 枠（ボーダー）の自動提案・手動調整
- Instagram・X（Twitter）への同時自動投稿
- PWA対応（iPhoneのホーム画面に追加可能）
- プッシュ通知

## 技術スタック
- HTML / CSS / JavaScript（PWA）
- Canvas API（画像編集）
- Puter.js（AI処理）
- Meta Graph API（Instagram投稿）
- X API（X投稿）
- Vercel（ホスティング）

## デプロイ
GitHubにプッシュするとVercelに自動デプロイされます。
